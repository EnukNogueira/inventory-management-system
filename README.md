# Sistema de Controle de Estoque

## Descrição
Desenvolvi este sistema em Java para gerenciar o controle de estoque, com foco específico no setor de manutenção industrial. O software organiza o fluxo de entrada e saída de materiais, visando garantir a integridade das informações e facilitar a consulta rápida de itens disponíveis.

## Tecnologias Utilizadas
* **Java (JDK 17+)**: Linguagem utilizada para implementar a lógica de backend.
* **Programação Orientada a Objetos**: Aplicação prática de conceitos como encapsulamento e herança para organizar o código.
* **JDBC/SQL**: Utilizado para a persistência e gerenciamento dos dados no banco de dados.

## Funcionalidades
* Cadastro técnico de itens e peças de reposição.
* Registro detalhado de movimentações de entrada e saída do almoxarifado.
* Monitoramento de níveis de estoque para auxílio na reposição de materiais.
